# BioEngine (archived — Triton-based, 2022–2024)

This is the original BioEngine built on NVIDIA Triton Inference Server. It was superseded in 2025 by a new implementation based on [Ray](https://www.ray.io/) and [Ray Serve](https://docs.ray.io/en/latest/serve/index.html), which supports multi-node GPU clusters, SLURM HPC, Kubernetes, and agent-readable interfaces.
